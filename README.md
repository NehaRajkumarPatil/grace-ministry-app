# 🌿 Grace Ministry Web App

A Progressive Web Application (PWA) for Grace Ministry to share sermons, devotionals, live services, and connect with users.

🔗 Live Demo: https://graceministryin.netlify.app/

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend/Services:** Firebase (Authentication, Firestore, Cloud Messaging)
* **PWA:** Service Worker, Web App Manifest
* **Deployment:** Netlify

---

## ✨ Features

* User authentication (Email & Google login)
* Admin dashboard for managing content
* Daily Bible verse (API + Firestore)
* Sermons and devotionals
* Live service integration (YouTube)
* Push notifications
* Responsive design with dark/light mode
